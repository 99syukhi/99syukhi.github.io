SnapBloom

A web photobooth that lets you pick your strip layout and decorate it however you like.

React Next.js CSS
GitHub

Overview

I started designing Snapbloom after coming across Nashallery's content on YouTube, TikTok, and Instagram. She has a few videos where she codes a photo booth, and I wanted to try that myself. Since I am still a student and still learning the ropes, I didn't build it entirely on my own; I designed it with the help of Claude. A draft exists, but it isn't a finished product yet.

The Idea

I wanted to create something fun, so I started by designing the logo and name. Next, I developed the idea further, considering how the photo booth should work and what the subsequent steps would be.

Features

  • Webcam capture directly in the browser
  • Multiple photobooth strip layout options
  • Download your finished strip as an image

Screenshots

What I'm Learning

Throughout SnapBloom, I'm learning how to work with browser APIs, manipulate images using the Canvas API, build reusable React components, and think beyond code by designing an experience that's both intuitive and enjoyable for users.
