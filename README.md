## About

This portfolio showcases who I am as a Software Engineering student, along with my skills, projects, and ways to get in touch.

The website is designed with a modern, responsive interface featuring a dark theme with subtle pink accents.

## Features

- Responsive design (mobile-first)
- Modern UI with custom styling
- Dark theme
- Lightweight and fast
- Pure HTML, CSS & JavaScript
- Hosted with GitHub Pages

## Running Locally

Clone the repository:

```bash
git clone https://github.com/99syukki/99syukki.github.io.git
```

Open the project folder:

```bash
cd 99syukki.github.io
```

Then simply open `index.html` in your browser or use a local development server such as the VS Code **Live Server** extension.
